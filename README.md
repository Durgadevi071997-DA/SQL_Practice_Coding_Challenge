

# SQL Practice Coding Challenge For Data Analytics




## Project Overview

This SQL practice repository is designed to simulate real-world business scenarios using customer, product, order, sales, employee, and salesperson data. It helps build strong SQL skills through practical queries focused on data analysis and reporting.

## Topic Covered


Database creation (DDL) and data manipulation (DML)

Joins (INNER, LEFT, RIGHT)

Aggregations and GROUP BY

Subqueries and CTEs

Window functions (RANK, ROW_NUMBER, running totals)

Basic indexing and query optimization

Business reporting queries

## Key Learning

Writing real-world SQL queries for data analysis

Extracting insights from multiple tables

Performing sales and customer analysi

Using advanced SQL techniques for reporting

Building strong Data Analyst–level SQL skills
